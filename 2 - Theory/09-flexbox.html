Flexbox is used to criate flexible div's.

Main Div

"display: flex;" or "flex-direction: row;"
[P1] — Let the div flexible in a row (in line)

"flex-direction: column;"
[P2] — Let the div flexible in a horizontal row
• Both can be reverse [column-reverse or row-reverse]


"justify-content: center;" — Change the items position (Flex-Start = Left // Flex-End = Right // Center)

"justify-content: space-between;" — Make exacts spaces between the options

"justify-content: space-around;" — Exacts spaces to fill all the screen

"align-items: center;" — Centralize (Vertical) the div items inside the main div
